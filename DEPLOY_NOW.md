## 🚀 Command to Push to GitHub

```powershell
cd "d:\foundation website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/restored-kings-foundation.git
git push -u origin main
```

> Replace `YOUR_USERNAME` with your GitHub username

## 📱 Then Deploy to Vercel

**Easiest Method (2 clicks):**

1. Go to: https://vercel.com/new
2. Choose: "Import GitHub Repository"
3. Paste: `https://github.com/YOUR_USERNAME/restored-kings-foundation`
4. Click: "Import"
5. Settings:
   - **Framework:** "Blank"
   - **Build Command:** (leave empty)
   - **Output Directory:** `public`
6. Click: "Deploy"
7. **Done!** ✓

Your site is live at: `https://restored-kings-foundation-xxxx.vercel.app`
